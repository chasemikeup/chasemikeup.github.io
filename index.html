You see, people run for different reasons. Some run to be a bit healthier, while others run because it's the only way to go to school.

I run because I want to prove myself that I can. I run to think about everything, about nothing. I run because when I do, I feel like I can run forever without stopping, although this feeling doesn't last that long.

I run

  • in the morning
  • during the time in between classes
  • before the sun completely rises
  • after the sun completely sets

Also, I run to encourage people to do so, mainly on Instagram
